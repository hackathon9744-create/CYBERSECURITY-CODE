# CYBERSECURITY-CODE
code of domain related to cybersecurity for pkp hackathon
A lightweight web dashboard that provides URL fraud checks, AI text-generation detection, and a mock news feed. Built with HTML, TailwindCSS, and Vanilla JavaScript, with optional Firebase logging.

Features

URL Fraud Check (pattern-based risk levels: Safe / Moderate / High)

AI Text Detection (probability score for AI-generated text)

Scrollable News Sidebar (mock data + refresh)

Optional Firebase Auth & Firestore logging for submissions
